# 产品规划

## 产品方向

Vibe Coding 作品广场第一阶段是一个轻量作品展示社区，重点解决三个问题：

1. 普通开发者可以展示自己用 AI 工具做出的项目。
2. 浏览者可以快速发现可体验、可咨询、可定制或开源的项目。
3. 平台可以通过提交表单收集作品，后续人工审核上架。

## 用户角色

- 浏览用户：查看作品、分类、项目详情和开发者信息。
- 作品开发者：提交作品、展示个人主页、表达是否接定制。
- 平台运营者：后续人工审核作品，本阶段不开发后台。

## 第一阶段功能优先级

### 必做

- 响应式基础布局
- 首页作品流
- 发现分类页
- 项目详情页
- 开发者主页
- 提交作品页
- 我的 / 关于说明页
- 本地 mock 数据

### 暂不做

- 用户账号
- 私信和即时聊天
- 服务履约闭环
- 后台审核系统
- 推荐算法

## 后续可能扩展

- 接入 Supabase 存储作品和开发者数据
- 增加登录与个人作品管理
- 增加收藏功能持久化
- 增加人工审核后台
- 增加搜索和筛选
