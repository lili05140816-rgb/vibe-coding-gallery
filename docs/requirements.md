# 需求说明

## 项目名称

Vibe Coding 作品广场

## 项目定位

这是一个专门展示普通人用 AI Agent / Codex / Claude Code / Cursor / Trae / Lovable 等工具做出来的小项目、AI 工具、工作流和网页应用的作品广场。

第一阶段目标是做轻量作品社区，不做重型商业平台。

## 第一阶段核心目标

- 展示作品信息流
- 展示作品分类
- 展示项目详情
- 展示开发者主页
- 支持提交作品表单
- 提供简单的“我的 / 关于”说明页

## 页面范围

1. 首页
2. 发现页
3. 项目详情页
4. 开发者主页
5. 提交作品页
6. 我的页面

## 响应式要求

- 移动端优先。
- 同一个网址自动适配手机和电脑。
- 手机端使用底部导航栏。
- 电脑端使用顶部导航栏。
- 首页作品卡片手机端单列，电脑端 3-4 列网格。
- 项目详情页手机端上下排列，电脑端左右分栏。
- 提交作品页手机端单列表单，电脑端居中宽表单。

## 第一阶段不做

- 服务履约系统
- 押金托管
- 需求大厅
- 即时聊天
- 复杂管理员后台
- 自动提现
- 原生 iOS / Android App
- 复杂推荐算法
- 真实数据库
- 真实登录注册
