# 测试清单

## 第一阶段基础测试

后续页面开发完成后，应至少检查：

- 项目是否可以正常启动。
- 首页是否能展示 mock 作品。
- 发现页分类是否能展示对应 mock 作品。
- 项目详情页是否能读取正确作品数据。
- 开发者主页是否能展示对应开发者和作品列表。
- 提交作品页表单是否能完成前端提交提示。
- 我的页面是否能正常展示说明内容。

## 响应式测试

- 手机端底部导航是否正常。
- 电脑端顶部导航是否正常。
- 手机端卡片是否单列显示。
- 电脑端卡片是否 3-4 列显示。
- 项目详情页手机端是否上下排列。
- 项目详情页电脑端是否左右分栏。
- 提交作品页手机和电脑端是否都不变形。

## 第一阶段禁用项检查

- 不应出现服务履约流程。
- 不应出现即时聊天。
- 不应出现真实登录注册。
- 不应连接真实数据库。
- 不应出现复杂后台。
