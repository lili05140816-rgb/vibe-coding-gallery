# UI 设计说明

## 设计关键词

- 简洁
- 年轻
- 科技感
- 作品社区
- 移动端友好
- 电脑端高浏览效率

## 视觉方向

- 使用浅色背景。
- 使用卡片式布局承载作品。
- 使用标签和状态徽章帮助快速识别项目类型。
- 使用适度圆角和轻量阴影。
- 可以使用渐变按钮，但避免过度复杂动画。
- 整体不要像传统商城，更像 AI 作品社区和灵感广场。

## 响应式布局

### 手机端

- 底部导航：首页、发现、发布、我的。
- 首页作品卡片单列展示。
- 项目详情页上下排列。
- 提交作品页单列表单。

### 电脑端

- 顶部导航：首页、发现、发布作品、关于我们。
- 首页作品卡片使用 3-4 列网格。
- 项目详情页左右分栏。
- 提交作品页使用居中宽表单。

## 页面设计要点

- 首页要能快速看到今日精选、最新发布、可体验项目、可咨询项目、接定制项目。
- 发现页用分类入口组织内容。
- 详情页突出项目卖点、截图、功能、技术栈和开发者。
- 开发者主页突出头像、昵称、技能、是否接定制和作品列表。
- 提交作品页要清晰、轻量，降低提交门槛。
